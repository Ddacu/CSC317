Assignment 3

Visit Assignment 3

This was a group Assignment where we all collectively work on a CSS of a portfolio page of one of the members. The Goal of this assignment was to showcase our understanding of CSS concepts, compare Manual and Ai CSS, Showcase the weaknesses of each of the styles. At the end create a combined CSS that takes the best elements from all the different styles. Finally, we are to show we were to demonstrate group collaboration.

  • Handwritten.CSS (Implemented a Layout that works for both Desktop and Mobile)
  • Ai.CSS (One For Each Member)
  • Hybrid.CSS (Combination of All CSS Styles)
  • Switcher.js (Used to switch between CSS Styles)
  • Github Workflow with Group Members
